# ファビコン・アイコン 制作仕様

サイトで使用する favicon および Apple / PWA 用アイコンの種類とサイズ一覧。

---

## 簡易版（これだけ作れば十分な場合）

**制作するのは次の2つだけ**で、多くのサイトは事足ります。

| # | ファイル         | サイズ   | 用途 |
|---|------------------|----------|------|
| 1 | **favicon.ico**  | 32×32    | ブラウザタブ・ブックマーク |
| 2 | **icon-180.png** | 180×180  | iPhone/iPad ホーム画面に追加したときのアイコン |

- **favicon.ico**: 32×32 の画像を .ico 形式で保存（オンライン変換ツールで PNG→ICO 可）。
- **icon-180.png**: 同じデザインを 180×180 の PNG で1枚用意。  
  → 192や512が必要な PWA は、この180を拡大して流用しても可。

PWA（ホームに追加・オフライン対応）を使わない場合は、**上記2点だけ**でOKです。

---

## 1. ファビコン（ブラウザタブ・ブックマーク）

| 用途           | サイズ   | 形式   | 備考                         |
|----------------|----------|--------|------------------------------|
| 汎用           | **32×32** | .ico   | 多くのブラウザで使用         |
| 汎用           | **16×16** | .ico に含める | タブ・ブックマーク用     |
| モダン向け     | **32×32** または **48×48** | .png | `<link rel="icon">` で指定可 |

- **推奨**: 16, 32, 48px を1つの **favicon.ico** にまとめるか、**favicon.ico**（32×32）＋ **favicon-32.png** など。

---

## 2. Apple Touch Icon（iOS・iPadOS ホーム画面など）

| 用途               | サイズ(px)   | 形式 | 備考                     |
|--------------------|--------------|------|--------------------------|
| iPhone             | **180×180**  | .png | iPhone Retina（推奨必須） |
| iPad               | **167×167**  | .png | iPad Pro など            |
| iPad（従来）       | **152×152**  | .png | 従来 iPad                |
| iPhone（非Retina） | **120×120**  | .png | 古い端末向け（任意）     |
| 汎用               | **180×180**  | .png | 1枚で済ませる場合の代表  |

- **最低限**: **180×180** の PNG 1枚（`apple-touch-icon.png` または `icon-180.png` など）。

---

## 3. PWA / Web App（Android・Chrome・マニフェスト用）

| 用途           | サイズ(px)   | 形式 | 備考                     |
|----------------|--------------|------|--------------------------|
| マニフェスト用 | **192×192**  | .png | Android ホーム画面・Chrome |
| マニフェスト用 | **512×512**  | .png | スプラッシュ・高解像度表示 |
| 任意           | 144, 384 など | .png | 端末ごとの最適化用       |

- **最低限**: **192×192** と **512×512** の2種。

---

## 4. まとめ：まず用意するとよいもの

| 種類               | サイズ       | ファイル例                    |
|--------------------|--------------|-------------------------------|
| ファビコン         | 32×32（＋16, 48 を .ico に含めても可） | `favicon.ico`          |
| Apple Touch Icon   | **180×180**  | `icon-180.png` または `apple-touch-icon.png` |
| PWA（小）          | **192×192**  | `icon-192.png`                |
| PWA（大）          | **512×512**  | `icon-512.png`                |

---

## 5. 本プロジェクトでの参照先

- **HTML head**: `_assets/_include_htmlhead.php`
  - `/favicon.ico`
  - `/icons/icon-180.png`（Apple Touch Icon）
- **PWA**: `manifest.webmanifest` で 192 / 512 のアイコンパスを指定する必要あり。
